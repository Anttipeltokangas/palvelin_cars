Autot CRUD ‑sovellus

Tämä on CRUD‑sovellus autojen hallintaan. Käyttäjä voi lisätä, muokata, poistaa ja hakea autoja. Sovellus on toteutettu HTML, CSS/Bootstrap ja JavaScriptillä. Taustalla toimii Node.js/Express‑palvelin ja MongoDB‑tietokanta.
