Harjoitustyö: Autot CRUD ‑sovellus

Tämä sovellus mahdollistaa autojen lisäämisen, muokkaamisen, poistamisen ja hakemisen. Käyttöliittymä on toteutettu HTML:llä, CSS:llä ja JavaScriptillä. Tiedot tallennetaan MongoDB-tietokantaan Node.js-palvelimen kautta.
